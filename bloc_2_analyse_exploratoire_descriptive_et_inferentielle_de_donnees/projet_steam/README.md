# Steam

Analyse globale de l'écosystème des jeux vidéo sur Steam pour identifier les tendances du marché et les facteurs de succès.

## Contexte & Objectifs

🚧 Ubisoft souhaite lancer un nouveau jeu vidéo révolutionnaire. Pour orienter la conception de ce titre, l'entreprise a besoin d'une analyse stratégique globale du marché sur la plateforme Steam afin de mieux comprendre l'écosystème actuel et ses tendances.

🎯 L'objectif de ce projet est de comprendre les facteurs qui influencent la popularité et les ventes d'un jeu vidéo à travers trois axes : une vue macro (éditeurs, prix, notes, sorties), une analyse par genres (représentation, rentabilité) et une analyse par plateformes de distribution.
