# Walmart Sales

Développement d'un modèle d'apprentissage automatique pour estimer les ventes hebdomadaires des magasins Walmart.

## Contexte & Objectifs

🚧 Le service marketing de Walmart souhaite concevoir un modèle d'apprentissage automatique capable d'estimer les ventes hebdomadaires de leurs magasins avec la meilleure précision possible. Cela permettra de mieux comprendre l'influence des indicateurs économiques sur les ventes et de planifier de futures campagnes marketing.

🎯 L'objectif de ce projet se divise en trois étapes : réaliser une analyse exploratoire (EDA) et le prétraitement des données, entraîner un modèle de régression linéaire de base, puis optimiser les résultats avec un modèle de régression régularisé pour éviter le surapprentissage.
