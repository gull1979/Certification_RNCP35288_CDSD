Le projet de fin de formation fullstack à pour thème :  l'agriculture mondiale<BR><BR>
Il a pour but de prédire le rendement des terres avant même que la récolte n'ait lieu.<BR>
A partir de facteurs mesurables : fertilisation, type de culture, structure des sols, pesticides.<BR>
L'objectif : transformer des données brutes en levier de planification stratégique.
